Algunos scripts basicos:

Ahora vamos a ver 2 scripts que usan los conceptos mas basicos de Unity, el de "GameObject" y el de "Transform", antes de nada explicar que son estas 2 cosas.

Transform:

Transform es un componente que tienen TODOS lo objetos en la escena, este es el componente que controla la posicion, rotacion y escala de dicho objeto dentro de la escena, a un objeto no se le puede quitar este conmmponente.

Game Object:

GameObject o gameObject es el propio objeto en si mismo, este "comando" sirve para señalar un objeto, como por ejemplo dentro del codigo se utilizaria GameObject para obtener un objeto y hacer cosas con el.
